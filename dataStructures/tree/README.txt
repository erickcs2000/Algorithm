This program is multitasking:

1) Generate a tree (Press N)
This function only inicialize the tree, but it is necesary to create the tree

2) Create an BST(Binary search tree) (Press L)

Here you have to put every number, and the program sort the tree

3) Search a number (Press F)

Find the leaf where the number is.

4) Print the tree (Press P)

5) Check if the tree is BST (Press B)

6) Check if the tree is AVL (Press V)

7) Add a number (Press A)

8) Delete a number (Press D)
