This program has some functions:

1) Inicialize a list (Press N)

2) Insert a number in a position (Press I)

3) Delete a number in a position (Press E)

4) Get the value of the number in a position (Press V)

5) Get the position of a number (Press B)

5) Printf the full list (Press T)

6) Modify a number (Press S)

7) Get lenght of the list (Press L)

8) Add more than one number at the end of the list (Press C)
